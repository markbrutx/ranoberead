# Ranobe Reader

Это проект для чтения ранобэ с веб-интерфейсом и API-бэкендом.

## Требования

- Python 3.8+
- Node.js 14+
- npm 6+

На Windows:

```
python -m venv venv
.\venv\Scripts\activate
```

На macOS и Linux:

```
python3 -m venv venv
source venv/bin/activate
```

```
pip install -r requirements.txt
```

```
npm run dev
```
